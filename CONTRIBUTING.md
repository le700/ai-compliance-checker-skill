# Contributing to AI Compliance Checker

Thank you for considering contributing! Your help makes this tool better for everyone.

## How Can I Contribute?

### Report Issues
- Use clear, descriptive titles
- Include steps to reproduce
- Provide relevant regulation references

### Suggest New Regulations
- Include official source links
- Provide implementation suggestions
- Consider cross-platform applicability

### Improve Detection Rules
- Add new prohibited content patterns
- Improve detection accuracy
- Add platform-specific rules

### Pull Requests
- Follow the markdown style guide
- Include tests for new detection rules
- Update documentation as needed

## Style Guides

### Git Commit Messages
- Use present tense ("Add rule" not "Added rule")
- Reference regulation numbers when applicable

### Markdown Style
- Use GitHub Flavored Markdown
- Include both Chinese and English where applicable
